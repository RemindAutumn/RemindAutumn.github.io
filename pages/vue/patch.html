# 📄遇到的问题

# favicon.ico图标

在vue 的脚手架中， favicon.ico图标不显示的问题

  • 方法1 : 查看/ build/webpack.dev.conf.js 文件中的配置是否正确
// build/webpack.dev.conf.js
new HtmlWebpackPlugin({

    filename: 'index.html',
    template: 'index.html',
    inject: true,
    hash: true,
    favicon: path.resolve('favicon.ico')

})
  • 方法2 : 将 favicon.ico 放在 static 目录下 , 手动写入
<!-- index.html -->
<link rel="shortcut icon" sizes="48x48" href="/static/favicon.ico">
