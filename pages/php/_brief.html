# 📄php（简）

php 标记 <?php 可以让代码进入PHP模式 ?> 可以让代码退出PHP模式

# 输出内容

以下是一些php常用的输出方式

# echo

    echo 'hellow world'
    echo 'hellow world','你好,世界'
    #可以使用多各参数
    #注意echo 后面跟着空格
    #该方法打印一个结果的时候,如果是false 则打印空

# print

    echo 'hellow world'
    echo 'hellow world','你好,世界' ❌
    #只可以使用一个参数
    #注意echo 后面跟着空格
    #该方法打印一个结果的时候,如果是false 则打印空

# var_dump

    var_dump('hellow world')
    # 在php中,该方法可以打印出 false 和true

在php中, 与js 不同的是, false 输出的结果是'' , true 的输出结果是 1
我们可以使用var_dump 才可以返回bool值 true 或者 false

# isset()

判断是否声明, 并且值是否为null

$age;
$name = '名字';
$sex = null
isset($name) //true
isset($age) // false
isset($sex) // false

# empty()

判断一个变量的值是否为空, 以下情况会被判断为空:
"" , 0 , "0" , null , false , array()

$age;
empty($age)//
