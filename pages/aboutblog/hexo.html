# 📄Heox

这是一篇比较傻瓜式的教学...

# 需要提前下载的一些东西 😀

  • 一个代码编辑器 比如 vscode (下载地址)
  • node.js LTS 版本(下载地址)

# 对应步骤

  • 安装 vscode,
  • 安装 node.js,
  • 打开 vscode,进入命令行:快捷键**Ctrl+ ** ( 就是 Ctrl+Esc 下方,Tab 上方的那个按键) 😄 没有这么傻瓜的! 不过 node.js 和 vscode 还是要安装好的, 安装完成之后可以通过npm -v` 看到 node 的版本

# 1.安装 hexo

npm install hexo-cli -g
// 全局安装hexo脚手架
hexo init blog
//新建一个名为blog的模板
cd blog
//进入blog文件夹
npm i (npm install)
//下载相关的包
hexo s (hexo server)
// 本地启动

至此你可以在你的本地看到你的博客了.

# 2.跟换 Icarus 主题

鼠标选中命令行之后. 通过快捷键Ctrl+C,然后输入 Y 确认终止运行. 在确认终止之后,输入以下代码

git clone https://github.com/ppoffice/hexo-theme-icarus.git themes/icarus

通过以上代码,我们将 Icarus 克隆到你的 themes 文件夹里, 完成后,你应该可以在你的 themes 文件夹中看到一个 Icarus 文件夹,如果你看到了,那么说明你就成功了.

然后修改我们的设置文件,在 blog 目录下的 _config.yml文件 找到 theme 行将其修改为

theme: icarus

修改完成后,Ctrl+S保存

重新在命令行输出 hexo s 重启本地服务器
