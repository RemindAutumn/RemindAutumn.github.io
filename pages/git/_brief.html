# 📄Git 命令（简）

首先需要确认你已经安装了 Git网址

在我们安装完成之后， 首先我可以通过 git --version 命令查看你的安装版本 如果可以看到控制台打印出的版本， 那么说明你的git 已经安装ok拉

# 开始简单的配置你的git

首先， 我们需要配置用户名和邮箱

# 我们在确认了版本之后,要设置你的用户
# 设置你的用户名称
git config --global user.name 'yourName' 
# 设置你的用户邮箱 
git config --global user.email 'yourEmail'

当我们配置好邮箱和用户名之后， 我们可以用如下命令查看

# 查看用户名
git config --global user.name
# 查看邮箱
git config --global user.email

# 设置 SSH Key

在我们设置好了用户名和邮箱中， 我们还需要生成一个 SSH Key

# 生成SSH Key  (注意，该代码请在cm的中运行，bash中不可以)
ssh-keygen –t rsa –C "你的邮箱@xx.com"
# 上面的命令输入完成之后会现实一些选项 ,一路回车直至完成.
# 最终,我们生成的 SHH key密钥在

生成Key时弹出选项， 回车选择默认即可。 Key保存位置： /root/.ssh

登陆GitHub, 创建new SSH key, 其内容为/root/.ssh/id_rsa.pub中文本
